readChar.R

@mwk · Nov 3, 2017 · 1 min read

Function of the day

When I read text into an R session, I typically use either base::readLines() or—the tidyversion analog—readr::read_lines(). Although I’ve long been aware of readChar(), I’ve managed to avoid learning what it actually does.

Function

This post introduces the readChar() function.

x <- help("?", "utils")
print(x)

Example

x <- example("readChar", package = "base", give.lines = TRUE)
tfse::map_chr("message", x)
